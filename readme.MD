# YouTube Video Downloader API

Backend API for YouTube video downloader with Buzzheavier unlimited storage.

## Features
- Download YouTube videos in all qualities
- Extract audio (MP3, WAV)
- Unlimited downloads via Buzzheavier
- Files never expire
- No registration required

## API Endpoints
- `POST /api/info` - Get video information
- `POST /api/download` - Download and get link
- `GET /api/stats` - Download statistics
- `GET /health` - Health check
